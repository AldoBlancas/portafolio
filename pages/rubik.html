Sistema Web para Torneos de Rubik

Este sistema fue desarrollado para organizar torneos de cubo Rubik, permitiendo registrar participantes, gestionar rondas, capturar tiempos, visualizar rankings y administrar los eventos desde una plataforma centralizada. Es ideal para competencias locales o escolares.
